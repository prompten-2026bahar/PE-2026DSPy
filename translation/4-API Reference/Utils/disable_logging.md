## dspy.disable_logging

```python
dspy.disable_logging()
```

DSPy genelindeki olay günlükleme (event logging) API'leri (`eprint()`, `logger.info()` vb.) tarafından kullanılan `DSPyLoggingStream`'i devre dışı bırakarak sonraki tüm olay günlüklerini susturur.

*Kaynak kod: `.venv/lib/python3.14/site-packages/dspy/utils/logging_utils.py`*
